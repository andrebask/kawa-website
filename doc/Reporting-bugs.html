Reporting bugs

To report a bug or feature request for Kawa (including Qexo or JEmacs), it is best to use the bug-submission page. You can browse and comment on existing bug reports using the Kawa Bugzilla page.

When a bug report is created or modified, mail is automatically sent to the <bug-kawa@gnu.org> list. You can subscribe, unsubscribe, or browse the archives through the bug-kawa web interface.
